3D Object Scanner

Rotating Around Objects to Get a Full View

For a two week project for my Principles of Engineering class, a friend and I created a 3D scanner. We rotated an object on the turntable, while tilting an infrared sensor along the vertical axis of the object. We created the mechanical structure of the 3D scanner using scrap wood and a laser cutter and two servos and wrote code to interpret the infrared sensor. In the end, the data wasn’t extremely clear, but we were able to get some reasonable images.
